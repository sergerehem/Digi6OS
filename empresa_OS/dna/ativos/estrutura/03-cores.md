# 03 — Paleta de Cores

## Cores da marca

### Profundo — #1E1D2A
- **HEX:** #1E1D2A
- **RGB:** 30 · 29 · 42
- **HSL:** 244° · 18% · 14%
- **Uso:** Cor base de fundo, superfícies, backgrounds escuros

### Azul Primário — #1A26F3
- **HEX:** #1A26F3
- **RGB:** 26 · 38 · 243
- **HSL:** 236° · 90% · 53%
- **Uso:** Cor principal de marca, CTAs, destaques, links, elementos interativos

### Verde Neon — #02F28C
- **HEX:** #02F28C
- **RGB:** 2 · 242 · 140
- **HSL:** 155° · 98% · 48%
- **Uso:** Cor de acento, confirmações, sucesso, energia, elementos de destaque secundário

## Combinações aprovadas
- Texto claro (#F0EFF8) sobre fundo escuro (#1E1D2A)
- Texto branco (#ffffff) sobre Azul Primário (#1A26F3)
- Verde Neon (#02F28C) como acento sobre fundos escuros
- Gradiente: #1A26F3 → #02F28C para elementos de destaque máximo

## Regras gerais
- Nunca usar as 3 cores com igual peso na mesma composição
- O Verde Neon é acento — não cor principal
- O Azul Primário é a cor de identidade da marca
- O Profundo (#1E1D2A) é para fundos, nunca para texto
