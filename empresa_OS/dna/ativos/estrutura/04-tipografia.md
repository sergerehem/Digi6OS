# 04 — Tipografia

## Fonte principal: Inter
- **Tipo:** Sans-serif humanista
- **Origem:** Google Fonts (gratuita, open source)
- **Link:** https://fonts.google.com/specimen/Inter
- **Uso:** Todos os textos da marca — títulos, corpo, labels, UI

## Fonte técnica: JetBrains Mono
- **Tipo:** Monospace
- **Uso:** Valores técnicos (HEX, RGB), códigos, labels de dados, rótulos numéricos

## Pesos utilizados
| Peso | Valor | Uso |
|------|-------|-----|
| Light | 300 | Textos de apoio, legendas longas |
| Regular | 400 | Corpo de texto, parágrafos |
| Medium | 500 | Labels, tags, textos de interface |
| SemiBold | 600 | Destaques de corpo, subtítulos menores |
| Bold | 700 | Subtítulos, cabeçalhos de seção |
| ExtraBold | 800 | Títulos principais, displays |
| Black | 900 | Elementos de destaque máximo (uso pontual) |

## Escala tipográfica
| Nome | Tamanho | Peso | Tracking |
|------|---------|------|----------|
| Display | 72px+ | 800 | -0.04em |
| H1 | 48px | 800 | -0.03em |
| H2 | 36px | 700 | -0.025em |
| H3 | 24px | 700 | -0.02em |
| H4 | 20px | 600 | -0.01em |
| Body | 16px | 400 | 0 |
| Small | 14px | 400 | 0 |
| Label | 12px | 500 | +0.08em |
| Mono | 13px | 400 | 0 |

## Regras
- Títulos: sempre Inter com tracking negativo
- Corpo: Inter 400, line-height 1.6–1.7
- Labels/tags: Inter 500, uppercase, tracking positivo
- Valores técnicos: JetBrains Mono
